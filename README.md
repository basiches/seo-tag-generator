# 🤖 מחולל תגיות SEO - Powered by Claude AI

כלי SEO חכם שמייצר תגיות HTML אופטימליות בעברית בלחיצת כפתור.

## מה הכלי עושה?

מכניסים מילת מפתח → Claude AI מייצר:
- **Title** (50-60 תווים) — עם מילת המפתח ושם המותג
- **Meta Description** (150-158 תווים) — משכנע עם CTA
- **H1** (עד 70 תווים) — כותרת ראשית מאופטימיזית

כל שדה מגיע עם:
- מד אורך ויזואלי (ירוק/צהוב/אדום)
- כפתור העתקה מיידי

## טכנולוגיות

- **React** — ממשק משתמש
- **Claude API** (claude-sonnet-4-20250514) — מנוע AI
- **Heebo Font** — טיפוגרפיה עברית

## שימוש

```bash
npm install
npm run dev
```
